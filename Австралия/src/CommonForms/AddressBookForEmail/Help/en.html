With the address book, you can browse through the contacts and select email recipients.

For a better user experience, all application contacts are stored in the single address book.

To open the address book, place the cursor to one of the email recipient selection fields (To, Copy, or Bcc) and click Select.

The address book is divided into two parts:

  • The left side contains the following tabs:
      • Search contacts. In this field, enter a search request to find a contact by:
          • Email.
          • Email server domain.
          • Keywords in the contact description. It is a full-text search that looks through all contact attributes.
          • Contact description leading characters (Begins with).
          • In the search window bottom, you can see Refresh button, which is active when the search index is outdated.
        • Select a search option, enter a search request, and click Find. The search result will contain the contacts that meet the conditions.
      • Users. Select a contact from the application user list.
        • The tabs can contain another lists. For example, a list of partners.
        • For a faster browsing, use Find. The search result list supports all regular commands and actions.
      • Contacts by subject. The tab is active if the email Subject is filled in. The tab contains the list of contacts related to the subject. The list includes contacts that are mentioned in the emails and interaction documents with this subject.
    • Double-click on a contact to add it to the recipients. Also, to add or remove contacts, click on >, <, or <<.
  • The right side of the address book contains the list of email recipients.
    • The list includes the following columns:
      • In Group you can specify what email field you add contacts to. The following options are available: To, Copy, Bcc, and Reply to. The default field is To.
      • Address contains contact's email. If a contact is missing an email, you can type it to this column.
      • Presentation contains contact's description.
    • The right-side context menu contains the following commands: Find in list, Find by address, Find by presentation, Set contact address.
  • If a contact is not found, you can Add it to the address book. New contact requires description and email. Also, you can add a line to the list by pressing Ins.
  • With commands under More actions, you can Remove contact and move contacts up and down the list.

Adding contacts to email recipients

  • To add a contact to a recipient field, select the field and click Select, or double-click on the field.
  • To add multiple contacts, select all of them and click Select.
