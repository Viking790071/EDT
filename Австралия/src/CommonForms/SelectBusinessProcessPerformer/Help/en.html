Designed to select a business process performer (for example, Job).

Открывается из бизнес-процесса по кнопке выбора в поле Исполнитель.

В зависимости от специфики бизнес-процессов задачи могут быть адресованы исполнителюили группе исполнителей как персонально (персональная адресация - в качестве исполнителя можно выбирать конкретного пользователя), так и для некоторых бизнес-процессов с использованием ролей бизнес-процесса (ролевая адресация).

Если выбирается исполнитель, то бизнес-процесс будет адресован этому исполнителю. If you select a role as performed, the business process will be assigned to a user currently performing this role.
Roles can be reassigned at a later time. In this case, the task will be automatically reassigned to new performers.

Users

  • Выделите нужного пользователя, нажмите Выбрать
    • .
  • You can also double-click the item.

Peculiarities of interaction with external users

  • External users can be performers of some business processes (for example, partners or contact persons of partners).
  • As a rule, external users cannot use the names of specific employees of a company, thus, they can only specify the performer roles assigned to them.
  • To do that, you need to configure a list of such roles, specifying that they are designed for certain categories of external users.

Performer roles

  • In case of role addressing, assignment is not issued to the performer, but to a certain role of the business process (for example, Room supervisor).
    • Specifying, main, and additional addressing objects can be used. For example, the Project manager role can have the main Department addressing object and additional Project object.
    • Thus, addressing objects help to clarify to which project managers and in which department the task is assigned.
  • Select a performer role and click Select.
  • You can also double-click the item.
  • If necessary, select the addressing object. Click OK.
