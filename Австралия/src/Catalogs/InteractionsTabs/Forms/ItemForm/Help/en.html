This form is used to configure and save filters you intend to use for quick access to groups of interaction documents.

All tabs are personal. Tabs are used to group interaction documents. For example, you can group the documents by departments.

Adding interactions to bookmarks

  • Enter Description. The field is required.
  • To specify a new filter condition, click Add new item.There can be several conditions.
  • In the Field column, select the field of the interaction document to use for the filter. Select the field in the list and click Select , or double-click it.
  • In the Comparison kind column, select a comparison operation. The list includes both common (Greater or equal) and non-standard comparison operations:
    • Comparison operation In list is used to specify the list of allowed filter values. For example, it can be used to add several correspondence addresses of a partner to a list. Comparison operation Not in list is inversion of the above. For example, it can be used to move all emails to a folder expect emails sent by persons in the list.
    • Operations Contains and Does not contain are used to search for a field (name, address, or email subject) which you only know partially. For example, they can be used to identify emails incoming from contact persons of a specific partner if you know this partner's email domain. Any email sent from an address including this domain are considered to be sent by contact persons of this partner.
    • To find out whether a field contains a value, use the Filled and Not filled operations.
  • Specify Value of the filter field. Depending on the selected field and comparison type, select a value from the list, type in a value, or create a value list.
  • You can group the filter conditions if necessary. To do this, you need to apply logical AND, OR, or NOT to the selected conditions using the Group conditions command.
    • Select a condition, execute the Group conditions command, select a Group type: AND, OR, or NOT, and drag the other conditions to this group (or create conditions directly in the group).
    • If multiple conditions are used without grouping, logical AND applies to them by default.
  • To apply the filters, click Save and close.
