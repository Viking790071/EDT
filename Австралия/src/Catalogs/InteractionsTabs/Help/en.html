Tabs are used to group documents in interaction list. For example, you can group the documents by departments. All tabs are personal.

To enable tabs, switch the interaction list to By tabs grouping mode.

Creating tabs

  • To create a tab, right-click the grouping panel and execute the Create command.
  • Fill in the required fields.

Grouping tabs

  • Tabs can be grouped.
  • To create a tab group, right-click the grouping panel and execute the Create group command.
    • Enter group Description.
    • Select a User from the Users list to assign as the owner of the tab group . The current user is selected by default. Please keep in mind that the tab group and its content is only visible to the owner of the tab group.
    • The Group field is filled automatically if the tab group belongs to a parent group. The tab group and its content is only visible to the owner of the tab group. You may select another group if necessary.
  • Then, add or drag one or more tabs to the group.

Using tabs in the interaction list

  • To add a document to a tab, such as Favorite interactions, right-click the document in the Interactions list and execute the Add to tabs command.
  • Then, you may drag the tab to a tab group.

Using tabs in user profile

  • All actions with the user tabs also can be performed from the user profile.
  • For example, to access your tabs, execute the User details command in the action panel of the Main section.
