This list is used to apply email processing rules.

Generally, the rules apply automatically, but manual application may be required in specific situations. For example, upon creating a new rule, you might want to apply it to your earlier mail.

To open this list, click Apply rules in the Email processing rules window, or right-click the grouping panel in the interaction list (grouped by folder) and execute the Apply processing rules command.

The list includes the following columns:

  • Condition. The name of the condition.
  • Put in folder. The name of the folder to apply the rule to.

Applying rules

  • In the For emails in folder field, specify the folder the new rule will apply to.
  • To process the subfolders of this folder as well, select the Recursive check box.
  • Select check boxes for all the rules you want to apply. To select all rules, click . To clear all selection, click again.
  • Configure the rule application order. The rules at the top of the list are applied first. To reorder the rules, click Up and Down in the command panel of the list.
  • Click Apply. The selected rules will be applied to all email in the specified folder.
  • Messages about email processing progress are displayed at the bottom of the window.

Example

Two incoming email processing rules were created recently. Let us apply them to the unsorted email in Inbox folder.

  • Specify Inbox as the folder to process.
  • Clear the Recursive check box.
  • Select the two new rules in the rule list and click Apply.
