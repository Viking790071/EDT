Description of mail folder properties.

Each email account has six predefined folders that cannot be changed or deleted:

  • Inbox stores all incoming mail.
  • Outbox stores all outgoing mail that was not yet sent.
  • Junk mail stores all mail identified by mail server as junk (for more details, refer to the mail server documentation).
  • Sent items stores all sent mail.
  • Deleted items stores all mail marked for deletion.
  • Drafts stores mail to be edited before sending.

Rules for sorting mail between the folders are specified using Email processing rules. Rule management is available in the Email account navigation panel.

Creating folders

If the predefined email folders are not enough for your purposes, you can create custom folders. Personal folders can be created within the predefined folders. This is useful when you want to sort mail from a perdefined folder into several folders.

  • Select a predefined or personal folder where you want to create a new folder, if necessary.
  • Enter Description for the new folder.
  • If a folder was selected, Parent folder is assigned by default. Select a folder from the list if necessary
