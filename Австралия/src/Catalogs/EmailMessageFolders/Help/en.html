This catalog is used to sort the incoming and outgoing mail. Email account is the owner of the folders.

To open the catalog, execute the Email folders command in navigation panel of the email account. For convenient email folder management, enable grouping By folders in the interaction list.

Each email account has six predefined folders that cannot be changed or deleted:

  • Inbox stores all incoming mail.
  • Outbox stores all outgoing mail that was not yet sent.
  • Junk mail stores all mail identified by mail server as junk (for more details, refer to the mail server documentation).
  • Sent items stores all sent mail.
  • Deleted items stores all mail marked for deletion.
  • Drafts stores mail to be edited before sending.

Rules for sorting mail between the folders are specified using Email processing rules. Rules management is available in the Email account navigation panel.

Mail is sorted between folders automatically, upon status assignment or status change, or when processing rules are applied.

Creating folders

  • If the predefined email folders are not enough for your purposes, you can create custom folders.
  • Personal folders can be created within the predefined folders. This is useful when you want to sort mail from a perdefined folder into several folders.
  • To create a new folder, click Create and fill in the required fields.
