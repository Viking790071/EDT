With this form, you can create a template of an email or SMS.

Email templates

To create a new template, in the template list, select Create template. To create a template based on an outgoing email, select Generate from. In this case, the template will inherit some fields from the email. Fill in the following required parameters:

  • In the Description field, enter a friendly name.
  • Select the Template assignment:
    • Common for all types of documents and lists.
    • Only for to specify certain types of documents or lists.
  • Enter Subject. You can type plain text or text expression.
  • Enter message body text. In the body, you can type expressions. Select the expressions from Parameters, on the template right.
  • Body text supports formatting. The main format buttons are arranged in the toolbar above the text box. Все команды форматирования можно найти в меню Еще.
  • In the Parameters window, in the upper side of the template, you can see various attributes to insert a formula: Общие, а также поля выбранного документа. To add an attribute to the message Subject or body, drag and drop the attribute.
  • К сообщению можно прикрепить вложения с помощью команды Еще - Добавить в правой нижней части шаблона.
  • The selected print form formats are displayed below the Print forms and attachments window. Нажмите на ссылку, для того чтобы изменить список форматов.
  • To Archive print forms, select the check box.
  • Set the template availability:
    • In the Available field, select the users whom you want to grant access to the template. By default, templates are available to all users.
  • To Transliterate file names, select the check box. The names of report files will contain only Latin letters and digits. This ensures compatibility between different operating systems.

SMS templates

  • In the Description field, enter a friendly name.
  • Select the Template assignment:
    • Common.
    • Only for to specify certain types of documents or lists.
  • To Send in transliteration, select the check box.
  • Enter message body text. You can type plain text or text expression. In Parameters, on the right of the template, you find available attributes: document fields and Common attributes.
  • Set the template availability:
    • In the Available field, select the users whom you want to grant access to the template. By default, templates are available to all users.

Message generation

  • Select More — Generate message and select one of the following methods:
    • From template. It is the default method.
    • By external data processor. With this method, you can preview the message template. Highlighted fields will be populated.

Testing templates

  • After you created a template, you can test it to ensure that it works the way it should. To do so, select Test template.
