With this form, you can choose email and SMS templates.

The left part of the form contains template names. The right part contains template preview.

Generating messages

  • To generate a message from a template, select the template from the list. Then, select Generate.
  • If you want to send an SMS right away, select Generate and send.

Editing templates

  • To open a template for edition, double-click on it in the list. Or select the template and in the command bar select More — Edit.

Creating templates

  • The template list contains <No template> item.
  • To create a new template, double-click on the item. Or select the item and in the command bar select More — Edit.
