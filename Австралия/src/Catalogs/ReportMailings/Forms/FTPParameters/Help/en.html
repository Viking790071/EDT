Use this window to specify FTP connection settings for report distribution.
The Server and directory field is mandatory. Other fields are optional.

To open this window, in the report distribution rule set window, click Select.

Fill in the following fields:

  • Server and directory. A path to an FTP resource in the following format: ftp://server/directory/.
  • Port. The FTP port (the default value is 21).
  • Username. The username to access the FTP server.
  • Password. The password to access the FTP server.
  • Passive connection. Indicates whether FTP connection is passive.

Entering all parameters as a single string

  • Click Fill and enter a string in the following format: ftp://username:password@server:21/directory/. This will fill in all of the fields.
