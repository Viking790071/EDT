Use this list to select report recipients.

You can open the list by clicking Select recipients.

The list includes the following columns:

  • Recipient. A recipient or a group of recipients (as it appears in the appropriate list, for example, Users) and their email address.
  • Excluded. Use this check box to exclude a recipient from report distribution. You might need this option if you want to send reports to all group members except for a few: first select a group, and then exclude some of its members. You can also use this option to temporarily exclude some recipients from the list. Excluded users are marked red.

Selecting recipients

  • Click Select and then select recipients. You are prompted to select recipients from the list specified in the report distribution rule, for example, Users.
