Designed to monitor tasks.

Opened by the Current tasks command in the Task reports list of the Business processes and tasks section.

Report options

  • Current tasks. Tasks are grouped by performers in the report, the number and description of the task are displayed in the report strings, the deadline for completion is indicated. The report allows you to set a filter by completed tasks, the author, the date of actual and scheduled execution. Overdue and completed tasks can be marked in the report list. The report is generated for the Period.
    • If necessary, set the necessary parameters:
      • Business process state. Filter of active and stopped business processes.
      • Author. Filter of tasks of the specified supervisor.
      • Display completed tasks. Filter of completed tasks or uncompleted tasks.
      • Deadline. Filter of tasks that have due date before the specified date or the same date.
      • Execution date. Filter of tasks that have he same execution date as the specified one.
      • Select the check box to Mark overdue tasks.
      • Select the check box to Mark completed tasks.
  • Performance discipline help - the performer and the number of tasks for the specified period are displayed in the report string. Clarification about how many tasks are completed and uncompleted is given based on the tasks in the string. For completed tasks, the number of tasks completed on time and not on time is specified. For uncompleted tasks, the number of overdue tasks. By default, all tasks performed during the current month are displayed in the report.
    • The report is generated for Period. You can set the filter by Author and by the scheduled completion date (Deadline).
