Designed to monitor tasks and business processes. The report lists the hung tasks that cannot be performed because of the empty list of roles and performers.

If the Task control manager filter is set, the report will only display tasks assigned to performers in addressing objects supervised by a specified task control manager. You can set a filter by the Author task. There is also a filter by Period.

Report options

  • To group the list of hung tasks by addressing objects, select the Hung tasks summary report option. The report displays the main and additional addressing objects, responsible person, task. To reduce the information displayed in the report, clear Detailed check box. In this case, the description of the task is not displayed.
    • You can select tasks by Responsible person and by Author.
  • To view the number of hung tasks broken down by performer roles and addressing objects, select the Hung tasks by addressing object report option. The main and additional addressing object, task are displayed.
    • You can set a filter of hung tasks by a certain role performer, for example, by Task control manager.
  • To group the hung task by performer roles, for example, by Task control manager, select the Hung tasks by responsible person report option.
  • To view both the hung and overdue tasks, select the Overdue tasks report option.
    • Hung tasks are marked in red in the report (select the Mark tasks with no performers assigned check box on the report setting).
    • You can set a filter by a certain role performer, for example, by Task control manager.
