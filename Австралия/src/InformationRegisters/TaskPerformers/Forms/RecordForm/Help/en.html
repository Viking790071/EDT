Info about task performer with role-based addressing.

Opened by the Create command from the Role performers list.

Enter role performer

  • Using the button, select the list type: External users or Users.
  • Select a Performer of the role from the matching list.
  • In the Executes role field select a task performer role from the list (can be selected by default).
  • Role might require to select the main and additional addressing objects from the list.
