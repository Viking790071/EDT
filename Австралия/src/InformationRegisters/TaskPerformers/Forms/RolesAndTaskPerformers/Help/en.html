This list is used to display the latest information on map of users and performer roles.

Before you start working with business processes and tasks, it is recommended that you fill in the list.

To assign role performers that participate in business processes without addressing objects, execute the Roles and task performers command in the Administration - Organizer section.

To assign performers for roles that participate in business processes with addressing objects, execute the Roles and task performers command on the navigation panels of those objects, for example, on the Companies navigation panel.

The list includes the following columns:

  • Role. Role description. Roles with no assigned performers are marked with the Attention icon. This can result in hung tasks (tasks that can not progress because no users were assigned to perform them). The Hung tasks report is used to monitor any hung tasks. The Task control manager predefined role can be used to send hung task notifications.
  • Performers. A name of a performer (if one performer is assigned for this role) or a number of people assigned for this role.

Assign performers to role

  • Select a role, click Assign performers to role, select performers from the users list.
  • You can also double-click the item.
