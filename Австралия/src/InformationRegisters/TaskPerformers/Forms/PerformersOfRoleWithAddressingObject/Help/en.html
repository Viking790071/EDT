Designed to assign users to a performer role.

Opened by the Assign performers to role command from the Roles and task performers list.

At the bottom of the list, assigned users are counted.

Add performer

  • External users can be performers of some business processes (for example, partners or contact persons of partners).
  • To select one performer, click Add.
  • Using the button, select the list type: External users or Users.
  • Select a performer from the matching list.

Select assignees

  • To select one or several performers, click Select, select users from the list.
