It is used for recording interactions by a subject or a contact.

In the navigation panel of some documents or list items, for example, user lists, you can click Interactions to open interaction lists by a subject or a contact.

For example, if you click Interactions in the partner's contact person profile, all interactions the contact person is involved in are displayed. If you click Interactions for a document, all interactions by this subject are displayed.

You can access all Interactions in the interaction list.

The list includes the following columns:

  • Interaction type is shown as an icon:
    • Scheduled interaction
    • Meeting
    • Phone call
    • Text message
    • Incoming email
    • Outgoing email
  • Participants. Contacts that had interactions.
  • Date. The document creation date.
  • Subject. A short summary of the interaction.

In the list, documents for review, emails, and text messages with set flag Under control are highlighted in bold. Once you select the Reviewed check box, the highlighting is cleared.

Available view modes

  • The following view modes are available:
    • Tree. Interactions are displayed as a tree showing the structure of interaction chains.
    • List. Interactions are displayed as a flat list.
  • To change the view mode, click Switch view mode.

Filters
  • Using the Status field, you can display interactions with a particular review status:
    • By default, All interaction documents are shown.
    • If the filter is set to To review, the result will contain interactions that meet the following requirements: to be processed before the current date and Reviewed is cleared.
    • If the field is set to Deferred, the result will contain interactions with the filled Review after field and with the Reviewed check box cleared.
    • If the filter is set to Reviewed, the result will contain interactions that meet the following requirement: Reviewed is selected.
  • To filter the list by person in charge for processing interactions, click on Person responsible and select a user from the list of users.
  • To filter the list by interaction type, click More - Interaction type and select the type. For example, Email or Phone call.
  • To select a period of viewing interaction documents, click More - Set period.

Entering interaction documents

  • Click Create, select the required document and fill in the required fields.
  • If list is output by subject, the Subject field of a new interaction document is automatically populated with the same subject.

Setting the Reviewed flag for multiple documents
  • Select documents that require setting the Reviewed flag (remove the Under control flag for outgoing emails and SMS messages).
  • Click Mark as Reviewed.
  • Selection in the list is removed from reviewed interaction documents.

Removing the Reviewed flag for multiple documents
  • Выделите документы, у которых необходимо снять признак Рассмотрено (для исходящих писем и сообщений SMS - установить признак На контроле).
  • Use More - Clear the reviewed flag.
  • Нерассмотренные документы взаимодействий в списке выделяются жирным шрифтом.

Deferring document reviews
  • Выделите документы, рассмотрение которых необходимо отложить.
  • Click and specify date and time, to which you need to Defer review.

Назначение ответственного для нескольких документов

  • Select documents that require setting a responsible person.
  • Click Set responsible person.
  • Select a user from the list.

Setting interaction subject for multiple documents
  • Select documents that require setting an interaction subject.
  • Click Set interaction subject.
  • Select the document type from the list.
  • Select a document as an interaction subject from the corresponding list
