Set up an email signature

When setting up email operations, you can create signature templates for outgoing emails. Also signature templates can be created when setting up email accounts.

  • You can create a signature of two kinds:
    • for new messages using the Signature for a new email tab.
    • for messages when replying or forwarding using the Signature on replying tab.
  • Select the Enable check box for the signature that you need. Choose Signature format:
    • Plain text.
    • HTML. Creation of a signature in HTML format allows you to format text and insert pictures in it.
      • You can create signatures in HTML format if it is provided by the application settings. You can change this setting in the Administration - Organizer section by editing the Outgoing emails in HTML format check box value.
  • Write a signature to the mapping field. При выборе формата HTML можно выбирать различные способы форматирования текста, вставлять картинки и гиперссылки. You can format text using formatting buttons of navigation panel and also using the More menu commands.
Important. If a signature is set for an account, an account signature is inserted when creating an email.

Delivery and read receipts

On the Track emails tab you can enable the ability to request receipts from the recipient that sent emails were read and received:

  • Select the Read receipt and Delivery receipt check boxes.
  • This values will be used for each outgoing email as default values, but you can change them if you like.

Set up the order of responses to read receipt requests

You can set them up on the Track emails tab. The application has the following behavior options when getting read receipt requests (select one of the options using the radio button):

  • Always send notification. After you select the Reviewed check box in an incoming email, a read receipt will be sent.
  • Never send notification. Requests on sending a read receipt will be ignored.
  • Ask before sending notification. When you select the Reviewed flag in an email, the application will ask whether a read receipt needs to be sent.

Display base email body in the Outgoing email document when replying or forwarding
  • Select the Show original email body on replying or forwarding check box on the Other tab.
  • In this case, base email body is displayed only in the Outgoing email interaction document, it will not be included in the email being sent.
  • This value will be used for each outgoing email as the default value, but you can change it in the email itself.

Include base email body in the outgoing email body when replying or forwarding
  • Select the Include original email body on replying or forwarding check box on the Other tab.
  • This value will be used for each outgoing email as the default value, but you can change it in the email itself.

Set up sending messages in file mode at once

  • The file mode has the Send emails at once feature, emails are sent right after you click Send (by default emails are sent on schedule of the Receive and send emails scheduled job).
  • Also you can send emails in file mode by using the Send and receive command of interaction list.
