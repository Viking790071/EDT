This form is used to view and edit parameters of incoming and outgoing email.

To open the form, open an email and execute the corresponding command in the More menu.

Email parameters:

  • Created on. The date and time the editing of the email was finished (for outgoing email), or the date and time the email was sent (for incoming email).
  • Internal number. The registration number assigned to the email in the application.
  • Sent on. The date the email was sent. Displayed only for outgoing email.
  • Received on. The date the email was received. Displayed only for incoming email.
  • Include original email body. This check box can be modified for emails with Draft or Outgoing status. When this check box is selected, text of the original email is automatically attached to the outgoing email when sending.
  • Folder. Current folder of the email. Can be modified. To edit this parameter, click Show all and select a folder from the list.
  • Request delivery receipt. Indicates whether the delivery receipt is requested. This check box can be modified for emails with Draft or Outgoing status.
  • Request read receipt. Indicates whether the read receipt is requested. This check box can be modified for emails with Draft or Outgoing status.
  • Internet headers. Technical information for an incoming email.
  • IDs. Technical information for an outgoing email.
