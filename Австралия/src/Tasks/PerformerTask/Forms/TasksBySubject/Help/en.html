The list is used to view tasks created for an application object.

To open the list, select Tasks in the navigation panel of the application object.

Overdue tasks are highlighted in red. The completed tasks are inactive and greyed out.

The list includes the following columns:

  • Importance. Takes Normal (default), Low, High values. Use this field to specify that execution of a business process can be postponed or, to the contrary, should be accelerated.
  • Stopped business processes are marked with icon in this column. To Stop or Resume a business process, select a corresponding button.
  • Task. Full description of the task. Task icon is displayed next to it.
  • Task Assignee.
  • Due date. Date and time by which Job is to be completed.

Viewing a task

  • To open a task for viewing or editing, click Change,
  • or double-click it.
