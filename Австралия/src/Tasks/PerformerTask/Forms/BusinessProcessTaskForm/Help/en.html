Task information.

Execution start date

  • In the Start date field, specify the start date and time of task execution.
  • To make this field editable, select the Date of task start check box in the Administration – Organizer section.

Importance

  • The Importance field is used to specify that execution of a business process can be postponed or, to the contrary, should be accelerated.
  • Valid values for this field are Low, Normal (default), and High. Each importance level is marked with an icon in the task lists.

Execution result

  • To mark a task as completed, click Completed.
  • Date of completion is filled in automatically.

Forward to another performer

  • To forward a task, click Forward and specify the assignee.

Подчиненные бизнес-процессы

  • A new business process, for example, Job, can be created on the basis of this tasks by selecting Create on basis. This requires access rights.
  • Для того чтобы использовать эту возможность, должен быть включен флажок Подчиненные бизнес-процессы в разделе Администрирование - Органайзер.

Reminder

  • Для того чтобы не забыть выполнить задачу, можно ввести Напоминание, текст которого будет выведен в указанное время.
  • Click , select an action, for example, Remind, fill the required fields.

Additional information

  • To view additional task information, click More – Additional information.
