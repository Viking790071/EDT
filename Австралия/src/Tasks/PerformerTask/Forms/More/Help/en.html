This form is used to view summary task information. This information cannot be edited.

To open the form, click More – Additional information in the task window.

Task information

The form displays information common for all task kinds, including:

  • Task. The name of the task.
  • Number. Assigned automatically.
  • Written. The date and time of task creation. FIlled in automatically.
  • Deadline. The planned date and time the task must be completed.

Task is addressed to

  • Исполнитель - пользователь программы, назначенный исполнителем;
  • Assignee role. Selected from the Roles and assignees list.
    • Main and additional addressing objects. Information about the application objects that specify role addressing.

    Бизнес-процесс

    • Business process. Information about the business process (name, date, and kind) the task belongs to.
    • текущая Точка маршрута бизнес-процесса (точка карты маршрута бизнес-процесса, например, Выполнить или Проверить).

    Task execution result

    Attributes of this group are filled for completed tasks only:

    • Completed check box
    • Completed. Task completion date and time.
