The My tasks list is where the task performer activities are carried out. It is located on the application desktop. The list contains tasks assigned to the current application user.

Tasks not yet accepted are marked in bold. Overdue tasks are highlighted in red.

The list includes the following columns:

  • Task. Full description and author of the task.
  • Written/Deadline. The date and time of task creation, and the planned date and time of Job completion. To have time displayed for the dates in this list, select the Date and time in task deadlines check box in the Administration – Organizer section.
  • Start date of task execution. Filled in by the assignee. If not filled, the current date is selected automatically. To have this field editable, select the Date of task start check box in the Administration – Organizer section.

Viewing a task

  • To view a task, double-click it,
  • or select the task and click Open,
  • or right-click the task and click Open task.

Viewing a subject

  • Tasks can be created on the basis of application objects. To view an object that was used to create a task (the task subject), right-click the task and click Open task subject.

Viewing a business process

  • Each task is a step of a business process, such as Jobs. To view the business process a task belongs to, right-click the task and click Open business process.

Grouping

  • Long task lists can be grouped for user convenience. Click Group by to group a task list by:
    • Due date
    • Importance
    • Route point
    • Author
    • Task subject
  • To cancel grouping, click Group by – No grouping.

Accepting for execution

  • Tasks not yet accepted are marked in bold.
  • Use the matching button to accept a task For execution. In the case of role addressing, the task accepted for execution disappears from the tasks lists of other performers who are assigned to the same role.
  • To cancel execution of a task accepted earlier, right-click the task and click Cancel accepting the task for execution.

Execution

  • To mark a task as completed, click ,
  • or right-click the task and click Mark task as completed.
  • The completed tasks are automatically removed from the list.

Forward to another performer

  • Click Forward and specify a new assignee.
