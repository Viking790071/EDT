The list is used to select a Planned interaction document. The document is intended for planning interactions with one or more contacts. Use this document to schedule an interaction while the exact interaction type (a telephone call, meeting, email, or SMS message) is yet unknown.

To open the list, choose the Planned interaction type for a field (for example, when filling the Subject field in an interaction document) and click .

The list includes the following columns:

  • Registration date. The date the document was created.
  • Number. Populated automatically.
  • Description. Describes the interaction details.
  • Subject. Describes the main point of the document. For example, "Customer claim." Subject is adopted from an application document or interaction document. If you didn't specify the interaction subject, the Subject field is populated with the current value, and the document starts a new interaction thread.
  • Participant list. The persons mentioned in the interaction documents.

    • Schedule interaction
    • Send SMS
    • Send email
