Information for viewing

  • From. Information about an email sender.
  • To. Information about main recipients of an email.
  • Copies. Information about additional recipients of an email.
  • Subject. An email subject. Filled by a sender.
  • Email. An email body. See the associated tab.
  • Attachment list. Files attached to an email. Attachment files can be opened or saved to the disk using the commands of the context menu of the attachment list.
  • Importance. Indicates whether the interaction is highly important or insignificant. Possible values are Low, Normal, and High. Set by an email sender.
  • Number. The registration number assigned to the email in the application.
  • Отравлено - дата и время отправки письма.

Viewing images in the email

  • Incoming email can contain unsafe items. These are the images downloaded from the Internet or embedded scripts. They are disabled by default for security reasons.
  • If such item enabling is allowed in the application, if there are any, the notice with a reference using which you can Display insecure content is output.
  • Using the Display insecure content reference, the images downloaded from the Internet will be enabled. Other insecure email items will not be enabled.

Changing document

Заполните необходимые поля:
  • Subject. Describes the subject of the interaction. For example, customer claims. If the interaction subject is not defined, the current document serves as a subject. It means that such document begins the interactions thread. When you click Interactions from the document, the new document inherits the interaction subject and main fields from the parent document.
    • To fill this field, select a data type form the list, then select a specific document from the list.
  • Person responsible. Select a person responsible for the interaction from the users list. By default, the application specifies a person responsible for the account for incoming emails.
  • Reviewed. Indicates that the interaction is reviewed and does not require further attention. Reviewed documents are excluded from selection in the interaction list. With the mapping settings, after selecting the check box, a read notification will be sent to the sender.
  • Review after. It must be filled if you plan to postpone review of this email until a certain date. Select one of the following values from the list: In 15 min, In 30 min, In 1 hour, In 3 hours, or Tomorrow. To set a custom date and time, click .
  • Comment. Additional information about an incoming email.

You can set the application settings to show or hide the Reviewed and Review later fields. You can change the settings in Administration - Organizer.

Creating an outgoing email based on an incoming email
  • Click a button or execute a command in the More menu:
    • Reply. Creates an outgoing email addressed to the sender of the original email. The outgoing email is included in the interaction chain.
    • Reply all. Creates an outgoing email addressed both to the sender and all recipients of the original email (specified in the To and Cc fields).
    • Forward. Creates an outgoing email without a recipient. The outgoing email includes the text and attachments of the original email.

The Specify contacts link
  • The application allows you to specify list items as interaction participants. For example, clients or suppliers.
  • When receiving an email, the addresses of the sender and recipients of the email are analyzed. If there is a unique map of the address of the sender and the client, the application maps the sender and the client in the email. This allows you to further get information about all interactions with the client.
  • If a unique map could not be found, for example, the address was not found or was specified for several clients, these data must be specified manually.
  • If at least one recipient or sender is not mapped, the Specify contacts link is displayed in the email. Click the link to map and update the email addresses contained in the email, contacts, and their current contact information in the application in the Save addresses to contacts window.
    • If the address and the contact were previously mapped, it will be displayed in the list. If the address and the contact were not previously mapped, or if it is necessary to map the address and anther contact, or to write the address in another field of contact information, in the Contact field you can select a contact to write the new email address. In the Contact email kind field you can specify the field to which the contact email address will be written.
  • If a contact email address is specified in the application, then, when you receive an email from such a contact, the contact is mapped to the address automatically.
  • If all contacts are mapped, then this link is not displayed. You can open the window of contact mapping by the More - Specify contacts command.

Email parameters. Changing email folder

  • You can find Additional information about the email by executing the More - Email parameters command.
  • To change the incoming email folder, execute the More - Email parameters command, and change the Folder field in the displayed window.

Changing email encoding
  • Execute the More - Change encoding command.
  • Select the required encoding form the list, and click OK.

Creating interactions based on an incoming email
  • Click Interactions in order to:
    • Schedule interaction
    • Schedule meeting
    • Send SMS
    • Make call

Viewing interactions generated from email

  • To view the list of interactions generated from this incoming email, click Interactions on the navigation panel.

Printing an email

  • Click Print.
  • In the Print window, specify printer settings, such as number of copies, paper size, and more.

Working with attachment list

  • If an email has attachments, then by right clicking the context menu you can:
    • Open. Open the attachment file using the mapping external application.
    • Save as. Save the file to a folder on the computer. Select a folder on the computer and click Save. The application displays a message about the successful saving of the file.
    • Properties. View cards of the attached file.

Saving incoming email to folder on computer

  • Execute the More - Save email to disk command.
  • Specify the path on the computer. The email is saved in the EML format.
